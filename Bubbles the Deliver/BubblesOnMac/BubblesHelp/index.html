  • Build Connection

    Build Connection

    Available Connection

    Connection is the foundation of this application.
    Devices connected to the same router is always ready to work.
    Devices connected with each other via Persona Spot will work, too.
    As long as your devices are in the same local network (LAN) then they work properly, you can find them in Peers.

  • >Deliver Features

    Features

    Deliver Features

    Sending text between your devices is extremely easy, you can then copy your text or send it via messages and emails.
    Deliver can send any size of files from any sources to any devices in your Peers list, you can preview files and save received images to gallery.
    History is shown newest on top, you can resend any message you used before or you can clear them. History is automatically cleared once app quitted.
    Every received file is stored in Files safely. You can use or delete them anytime you want.
